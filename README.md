# 🏥 CareTrust - Healthcare Corporate Template

医療・ヘルスケア業界のコーポレートサイト向けテンプレートです。
「クリーン＆誠実」をテーマに、安心感を与えるデザインと、Framer風の美しいアニメーションを実装しています。

**👉 [Live Preview](https://kotao-boop.github.io/corporate-medical/)**

## ✨ 特徴
- 白と淡いブルーを基調とし、清潔感を強調。
- すりガラス効果（Glassmorphism）を用いたナビゲーションバー。
- 信頼性をアピールするBento Grid形式の特徴紹介セクション。
- AI生成による高品質なヒーロー画像を標準搭載。
